# Phase 37A – Einstufungstest A1 bis C2

Ziel: Der Sprachkurs ist nicht mehr nur Lernbereich, sondern enthält einen simulierten Einstufungstest.

## Umfang

- 6 Niveaus: A1, A2, B1, B2, C1, C2
- 42 Testfragen gesamt
- 7 Fragen pro Niveau
- 5 normale Aufgaben pro Niveau
- 2 Sprechaufgaben pro Niveau
- Timer-Konzept: 45 Minuten
- Ergebnis: geschätztes Niveau A1 bis C2
- Lernempfehlung: empfohlenes Niveau direkt öffnen

## Bewertung

Die Auswertung ist eine Lernempfehlung und kein offizielles Zertifikat. Speaking wird auf Geräten ohne zuverlässige Web Speech ehrlich als Selbstbewertung gezählt.

## GitHub Pages

Das Modul ist statisch und benötigt kein Backend.
