# G54.47.14C · Backup und Rollback

Umgesetzt wurden reproduzierbare Source-, Runtime- und Rollback-Pakete mit SHA-256-Prüfsummen. Ein echter Firestore-Export kann optional eingebunden werden. Ohne echten Export wird ausdrücklich dokumentiert, dass keine Datenbanksicherung enthalten ist.

## Befehle
```bash
node tools/release-backup.mjs
node tools/release-backup.mjs --firebase-export=/PFAD/ZUM/EXPORT
node tools/verify-backup.mjs /PFAD/ZUM/BACKUP-ORDNER
```
