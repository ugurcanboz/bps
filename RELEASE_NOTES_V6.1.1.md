# V6.1.1 TopNav Sync Stable

Gezielter Fix für Mobile Top-Navigation nach V6.1.0.

## Behoben
- Top-Menü für Üben vollständig erweitert: Mix, Mathe, Logik, EDV, IT/FISI, Allgemeinwissen, Englisch, Konzentration, Route, Visual IQ, Sprint, Jogging, CTC, Elite.
- Aktiver Top-Tab wird nach jedem Render automatisch horizontal sichtbar gescrollt.
- Home-Quicklinks verlieren ihre TopNav-Synchronität nicht mehr beim Sprung in Practice.
- Highscore-Tabs Heute/Woche/Monat/Gesamt filtern jetzt den Cloud-Highscore-Kontext.
- Settings enthält nun Profil, Cloud, Audio, Daten und Framework konsistent als TopTabs.
- Doppelte Cloud-Karte im Highscore-Bereich entfernt.
- Mobile TopNav CSS gegen abgeschnittene Tabs gehärtet.
