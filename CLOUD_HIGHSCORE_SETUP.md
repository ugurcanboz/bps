# V3.5.1 Active Cloud Highscore Setup

Das Highscore-Board ist technisch aktiv. Für echte geräteübergreifende Daten brauchst du Supabase.

## 1. Supabase Projekt erstellen
- Neues Supabase Projekt öffnen
- Project URL und anon public key kopieren
- In `data/cloud-config.js` eintragen

## 2. SQL ausführen
Im Supabase SQL Editor einfügen und ausführen:

```sql
create table if not exists public.highscores (
  id bigint generated by default as identity primary key,
  player_name text not null,
  class_code text default 'default',
  device_id text,
  mode text,
  title text,
  percent integer not null check (percent >= 0 and percent <= 100),
  score integer,
  total integer,
  duration text,
  rank text,
  app_version text,
  created_at timestamptz default now()
);

create index if not exists highscores_class_score_idx
  on public.highscores (class_code, percent desc, score desc, created_at desc);

alter table public.highscores enable row level security;

drop policy if exists "highscores_select_public" on public.highscores;
drop policy if exists "highscores_insert_public" on public.highscores;

create policy "highscores_select_public"
  on public.highscores for select
  using (true);

create policy "highscores_insert_public"
  on public.highscores for insert
  with check (
    char_length(player_name) <= 32
    and percent >= 0
    and percent <= 100
    and char_length(coalesce(class_code,'')) <= 40
  );
```
