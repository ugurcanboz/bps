# START_HERE

Öffne `index.html` lokal oder lade den Ordner auf GitHub Pages/Netlify hoch.

Für Weiterentwicklung:
1. Änderungen bevorzugt in `js/src/` machen.
2. Danach ausführen: `node js/build/build-app-bundle.js`
3. Danach testen: `node --check js/app.js`
