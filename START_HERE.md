# Eignungstest-Trainer V7.1.0 Clean Rebuild Stable

## Start
1. ZIP in einen komplett neuen Ordner entpacken.
2. `VERSION_CHECK.html` öffnen.
3. Danach `index.html` direkt im Browser öffnen.

## Erwartete Anzeige
In der App muss sichtbar stehen: `Framework V7.1.0`.

## Temporär aktiv
Diese Version löscht bei jedem Refresh alte Browser-Caches und meldet vorhandene Service Worker ab. Das ist Absicht, um alte Desktop-PWA-Reste auszuschließen.
