# Release Notes V5.1.0 Module Stable

## Ziel

V5.1.0 ist kein Feature-Update, sondern ein Architektur- und Wartbarkeitsupdate. Die App wurde so strukturiert, dass neue Aufgabenbereiche gezielt gepflegt und erweitert werden können.

## Neu strukturiert

- Mathematik als eigenes Source-Modul
- Allgemeinwissen als eigenes Source-Modul
- IT/FISI als eigenes Source-Modul
- EDV Multi-Choice als eigenes Source-Modul
- Logik als eigenes Source-Modul
- Konzentration als eigenes Source-Modul
- Visual IQ als eigenes Source-Modul
- Route Memory als eigenes Source-Modul
- EDV Renderer getrennt unter Renderers
- Build-Reihenfolge dokumentiert und fest verdrahtet

## Stabilisiert

- Cache auf V5.1.0 hochgezogen
- Storage/Profil/Focus Keys auf v510 vereinheitlicht
- Build-Script prüft fehlende Source-Module
- CTC-Lohr Simulation erneut geprüft
- EDV Multi-Choice erneut geprüft
