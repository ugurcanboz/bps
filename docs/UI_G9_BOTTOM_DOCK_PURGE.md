# UI-G9 Bottom Dock Purge

Die Bottom Navbar ist vom alten früheren Tabbar-Klassen System abgekoppelt. Aktiver Dock: `#egtBottomDock` mit `.egt-bottom-dock`, `.egt-dock-btn`, `.egt-dock-icon`, `.egt-dock-svg`, `.egt-dock-label`.

Neue Bottom-Nav-Änderungen laufen ausschließlich über `renderTabBar()` in `js/ui-home-renderer.js` und `css/ui-nav-foundation.css`.
