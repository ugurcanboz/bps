# G25 – Dozentenfokus & Hilfebedarf-Filter

## Ziel
Der Admin- und Dozentenbereich wurde um Filterchips und eine priorisierte Hilfebedarfsliste erweitert.

## Umsetzung
- Statusfilter: Alle, Stabil, Riskant, Kritisch, Inaktiv.
- Admin filtert global über alle Teilnehmer.
- Demo Dozent A/B filtern nur innerhalb ihrer zugewiesenen Gruppe.
- Hilfebedarfsliste sortiert nach Priorität: kritisch, riskant, inaktiv, stabil.
- Klick auf Teilnehmer nutzt weiterhin die bestehende Balken-Profilkarte.
- Keine Änderung an Neon-Startseite, Login-Sheet oder iOS-X-Fix.

## Ergebnis
Das Portal ist stärker auf Lehrkraft-/Dozentenbedarf ausgerichtet: zuerst Hilfebedarf erkennen, dann Teilnehmerdetails öffnen.
