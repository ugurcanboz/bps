# G54.50.2 · UI-/UX-Konsistenz

Diese Phase ergänzt eine zentrale, zuletzt geladene Konsistenzschicht für die gesamte App. Sie vereinheitlicht Abstände, Radien, Touch-Ziele, Formkontrollen, Fokuszustände, Dialogbreiten, Icon-Größen sowie mobile und barrierearme Zustände, ohne fachliche Module umzubauen.

## Leitplanken
- Mindest-Touchziel 44 px
- Formfelder auf iOS mindestens 16 px Schriftgröße
- einheitliche Fokusdarstellung
- reduzierte Bewegung und Forced-Colors unterstützt
- vorhandene Modulstile bleiben erhalten; die Schicht normalisiert nur gemeinsame Grundelemente
