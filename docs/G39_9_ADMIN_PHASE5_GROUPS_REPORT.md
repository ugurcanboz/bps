# G39.9 Admin Phase 5 – Gruppenverwaltung

## Ziel
Der Gruppenbereich wurde vom einfachen Mischbereich zu einem eigenen Verwaltungsmodul ausgebaut.

## Umgesetzt
- Gruppen-/Kursübersicht mit KPI-Leiste
- Gruppensuche nach Gruppe, Dozent und Status
- Statusfilter: Alle, Stabil, Riskant, Kritisch, Inaktiv
- Gruppenliste mit Teilnehmerzahl, Durchschnittsquote und kritischen Profilen
- Gruppendetail rechts mit Dozent, Status, Kennzahlen und Risikoteilnehmern
- Teilnehmer aus Gruppendetail direkt ins Teilnehmerprofil öffnen
- Kurs-ID-Konfiguration, Teilnehmererstellung, Demo, Export und Reset im Gruppenbereich neu strukturiert
- Mobile Ansicht angepasst

## Hinweis
Die dauerhafte Speicherung von Dozentenzuordnung/Gruppenstatus ist vorbereitet, aber sollte in Phase 7 sauber mit Rechteverwaltung verbunden werden.
