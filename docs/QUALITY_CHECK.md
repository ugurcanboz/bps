# Qualitätscheck

- Nutzerbereich zeigt einen einheitlichen App-Namen.
- Admin-Funktionen liegen im Admin Portal.
- Teilnehmerverwaltung arbeitet über eine stabile userId.
- Passwort-Reset läuft über ein Einmalpasswort mit anschließendem Pflichtwechsel.
- Design folgt dem bestehenden Apple-/Glassmorphism-Konzept.
