# G23.17 iOS Close & Demo-Daten Fix

## Änderungen
- X-Schließen-Button bekommt robuste Touch-/Pointer-Hitbox und Capture-Handler.
- Portal kann auf iPad/iPhone weiterhin per Finger gescrollt werden, ohne den X-Button zu blockieren.
- Leere Teilnehmerliste erklärt lokale Gerätespeicherung.
- Admin kann Demo-Daten direkt aus der leeren Teilnehmerliste auf dem jeweiligen Gerät erzeugen.
- Neon-Startseite nicht verändert.

## Hinweis
Demo-Teilnehmer werden lokal gespeichert. PC, iPad und iPhone teilen diese Daten ohne Cloud/Synchronisation nicht automatisch.
