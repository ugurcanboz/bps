# G27 Aufgabenbank-Qualitätsprüfung

## Ziel
Admin erhält eine eigene Ansicht zur Aufgabenbank-Qualität. Die Prüfung verändert keine Aufgaben, sondern zeigt Struktur- und Qualitätsdaten.

## Neu
- eigener Admin-Tab **Aufgabenbank**
- Qualitätswert in Prozent
- Aufgabenanzahl
- kritische Punkte
- Hinweisarten
- Fachgruppen-Balken
- Schwierigkeits-Balken
- Fachregel-Hinweise
- Stichprobenliste
- Prüfbericht kopieren

## Geprüft werden u. a.
- fehlende IDs
- doppelte IDs
- leere oder zu kurze Fragen
- fehlende Antwortoptionen
- leere Antwortoptionen
- ungültiger Correct-Index
- fehlende Erklärungen
- fehlende Tags
- mögliche Dubletten
- Python außerhalb IT/FISI
- kaufmännische/pädagogische Fachregel-Hinweise

## Nicht verändert
- Neon-Startseite
- Login/Portal-Schließen
- Dozent A/B Rechte
- Teilnehmerprofile
- Hilfebedarf-Dropdown
- Systeminfo-Zählung aus G26.1
