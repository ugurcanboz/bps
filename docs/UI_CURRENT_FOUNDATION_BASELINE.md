# Aktueller UI-Foundation-Basisstand

Dieser Stand ist die aktuelle aktive UI-Basis.

Aktive UI-Kernschichten:

- `css/ui-foundation.css`
- `css/ui-nav-foundation.css`
- `js/ui-home-renderer.js`
- `js/ui-router.js`

Grundregel:

- Neue Kacheln nutzen `data-ui-action` und Foundation-Klassen.
- Neue Menüs/Sheets laufen über das zentrale Overlay-/Router-System.
- Alte UI-Patch-Schichten werden nicht weitergeführt.

Nicht berührt:

- Aufgabenbank
- Coach-DNA
- Admin-/Passwortsystem
- PWA-Grundstruktur
