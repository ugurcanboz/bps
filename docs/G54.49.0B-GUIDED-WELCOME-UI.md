# G54.49.0B · Novura Guided Welcome UI

## Ergebnis
Der Zugang wird nun als geführter, empathischer Mini-Dialog dargestellt. Die Oberfläche nutzt ausschließlich die deterministische Zustandsmaschine aus G54.49.0A.

## Ablauf
1. Stimmung auswählen
2. heutiges Ziel auswählen
3. Einloggen, Registrieren oder Demo starten

Ein Abbruch beendet nur den Dialog. Die drei Zugangswege bleiben sichtbar und bedienbar.

## Integrationsschutz
Die UI ersetzt keine Authentifizierungslogik. Nach der Auswahl delegiert sie an die vorhandenen Gate-Sheets und damit an `EGTAuthEngine`. Firebase-Sitzungen, Teilnahmecodes, Rollenprüfung und Demo-Zähler bleiben unverändert.

## Datenschutz
Stimmung und Ziel liegen nur im Speicher der geöffneten Welcome-Instanz. Beim Schließen wird `clearSensitiveContext()` ausgeführt. Es erfolgt keine Persistenz in LocalStorage, Firebase oder Profilen.

## Barrierefreiheit
- Dialogrolle und modaler Status
- sichtbarer Tastaturfokus
- semantische Buttons
- Live-Region für Schrittwechsel
- Reduced-Motion-Unterstützung
- mobile Safe-Area-kompatible Scrolloberfläche
