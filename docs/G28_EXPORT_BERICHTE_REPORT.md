# G28 Export & Berichte

## Ziel
Admin und Dozenten erhalten einen eigenen Bereich **Berichte**.

## Rollenlogik
- Admin exportiert alle sichtbaren Gruppen und Teilnehmer.
- Dozent exportiert nur die eigene zugewiesene Gruppe / sichtbare Teilnehmer.
- Kein Login: keine Berichte sichtbar.

## Funktionen
- CSV-Bericht
- JSON-Bericht
- Hilfebedarf-CSV
- Hilfebedarf-JSON
- Bericht kopieren
- Berichtsvorschau als Dropdown
