# UI-F1.3 Navigation Desktop/Mobile Fix

Ziel: Die Premium-Home-Startseite soll auf Chrome/Desktop nicht mehr durch eine überdimensionierte Bottom-Navigation verdeckt werden.

## Umsetzung
- Desktop/Chrome ab 781px: Bottom Navigation ausgeblendet, Schnellzugriff übernimmt Navigation.
- Mobile bis 780px: kompakte native App-Dock-Navigation bleibt erhalten.
- Content-Padding angepasst, damit keine Cards verdeckt werden.
- Premium-Banner bekommt Abstand nach unten.
- Keine Änderungen an Aufgabenbank, Coach, Admin oder Passwortsystem.
