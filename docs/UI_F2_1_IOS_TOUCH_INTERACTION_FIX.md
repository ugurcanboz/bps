# UI-F2.1 iOS Touch Interaction Fix

- Event-Delegation auf document
- touchend + pointerup + click + keyboard
- Doppeltrigger-Schutz
- Bottom-Tab-Touch-Support
- Kachel-/Button-Touch-Support
- SVG/Icon-Layer blockieren keine Taps mehr

Keine Änderung an Aufgabenbank, Coach-DNA, Admin- oder Passwortkern.
