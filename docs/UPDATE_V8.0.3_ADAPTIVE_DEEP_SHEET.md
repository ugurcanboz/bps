# Update V8.3.5 · Adaptive Deep Sheet

- Deep Sheet bleibt auf Desktop und Mobile aktiv.
- Sheet-Skalierung reagiert auf Breite, Höhe, Pointer/Touch und Orientation.
- Desktop erhält ein breiteres, sauber zentriertes Bottom-Sheet.
- Mobile erhält ein vollbreites, höheres Sheet mit sicherem Scrollbereich.
- Unterkategorien bleiben erhalten.
- Alte Desktop-Inline-Vorschau entfernt, damit das Deep Sheet wirklich das zentrale Menü ist.
- Sheet-DOM wird jetzt automatisch erzeugt, falls es im HTML fehlt.
- Serviceworker-Diagnose wird bei file:// zusätzlich in ArchitectureGuard übersprungen.
