# G54.50.2H · Rechtsdialog-Navigation und Rückweg

## Ziel

Rechtliche Dokumente und Informationsseiten dürfen keine Sackgasse bilden. Nutzer sollen eine Ebene zurückgehen können, ohne den gesamten Dialog schließen und den Weg erneut öffnen zu müssen.

## Umsetzung

- allgemeine Deep-Sheet-Unterstützung für einen optionalen Zurück-Button
- Zurück-Button neben dem Schließen-Button
- klare Trennung: Zurück = eine Ebene, X = gesamtes Sheet schließen
- Elternpfade im Informations- und Rechtscenter
- Wiederherstellung der Scrollposition pro Ebene
- mobile Darstellung als kompakter Pfeilbutton mit zugänglicher Beschriftung
- keine zusätzlichen Blur-, Filter- oder Daueranimationen

## Navigationspfade

- Nutzungsbedingungen → Rechtliches → Informationen
- Datenschutzerklärung → Rechtliches → Informationen
- Impressum → Rechtliches → Informationen
- KI-Hinweise → Rechtliches → Informationen
- Betreiber-Check → Rechtliches → Informationen

## Barrierefreiheit

- echte Buttons
- sichtbarer Tastaturfokus
- `aria-label="Zurück"`
- Touchziel mindestens 44 Pixel
- Reduced-Motion-kompatibel
