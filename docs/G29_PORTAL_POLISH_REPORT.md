# G29 Portal-Finalisierung & UX-Politur

## Ziel
G29 ist keine neue Feature-Version, sondern eine Politurversion für das bestehende Admin-/Dozentenportal.

## Umgesetzt
- Portalnavigation sprachlich geglättet: Dozentenblick, Admin Übersicht, System, Aufgabenbank, Berichte, Roadmap.
- Portal-Titel rollenabhängig: Login, Admin Portal, Dozenten Portal.
- Tabs werden abhängig von der Rolle verständlicher beschriftet.
- Leere Teilnehmerzustände sind erklärender und handlungsorientierter.
- Kachel-/Button-/Dropdown-Standard im Portal weiter vereinheitlicht.
- Mobile Darstellung der Portalnavigation weiter abgesichert.

## Nicht verändert
- Neon-Startseite.
- Login-/Rollenlogik.
- Dozent A/B Gruppenzugriff.
- Teilnehmerprofil-Balkenansicht.
- Hilfebedarf-Dropdown.
- Aufgabenbankprüfung.
- Export-/Berichte-Funktionen.

## Bewertung
G29 stabilisiert die Portal-UX und bereitet die nächste Phase G30 Berichte/Förderübersichten vor.
