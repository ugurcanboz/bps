# UI-E Repair · Legacy Design Firewall

Ziel: Alte helle/experimentelle Designreste dürfen nicht mehr sichtbar werden, auch wenn alte Renderer wie `.premium-card`, `.ghost`, `.bps-coach-*` oder Dashboard-Anhänge noch DOM erzeugen.

Umgesetzt:
- alte helle Premium-Karten auf dunkle Apple/Glass-DNA gezogen
- alte Ghost/Secondary/Danger-Buttons auf das UI-A Buttonsystem abgebildet
- alte Coach-Renderer optisch an UI-E angepasst
- Legacy-Kompatibilitätscontainer hart unsichtbar gehalten
- Mobile-Zwei-Spalten-Reste entschärft

Logik wurde nicht verändert. Aufgabenbank, Admin, Coach-DNA und PWA bleiben erhalten.
