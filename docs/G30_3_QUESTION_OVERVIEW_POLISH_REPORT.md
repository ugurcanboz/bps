# G30.3 Question Overview Polish

## Ziel
Die rechte Frageübersicht im Trainingscockpit wurde als kompakte, wiederverwendbare Statuskarte gestaltet.

## Änderungen
- Frageübersicht auf Desktop nicht mehr als riesiger leerer Kasten.
- Kompakte Question-Overview-Card mit Header, Zahlenchips und Status-Legende.
- Beantwortet = grün, aktuell = violett/blau, markiert = orange, übersprungen = rot.
- iPad/iOS behalten horizontale, fingerfreundliche Frageübersicht.
- Neon-Startseite, Adminportal und Teilnehmerlogik nicht verändert.
