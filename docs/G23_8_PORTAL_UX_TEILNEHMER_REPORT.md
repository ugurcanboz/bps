# G23.8 Portal UX & Teilnehmer-Begriff

## Ziel
- Dozentenbereich nutzt konsequent den Begriff **Teilnehmer** statt Lernende.
- Admin-/Dozentenportal bleibt auf dem funktionierenden Neon-UI-B-Stand.
- Portal-Kacheln werden als einheitliches, erweiterbares Grid abgesichert.

## Änderungen
- Sichtbare Texte im Dozentenbereich korrigiert: Lernende/Lernendendaten -> Teilnehmer/Teilnehmerdaten.
- Systeminfo und Roadmap auf G23.8 aktualisiert.
- Zusätzliche CSS-Regeln für upgrade-sichere Portal-Karten ergänzt.
- Keine Änderung an Neon-Hero, Startseite, Trainingskarten oder Bottom Navigation.

## Ergebnis
Das Portal bleibt visuell stabiler, Karten brechen automatisch um und spätere Erweiterungen sollen nicht mehr überlappen.
